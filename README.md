# Solar Portfolio

This is my personal portfolio. I built it with ReactJS & CSS. Hope you like it!

## Project Structure

- `public/` – contains static files like `index.html`, favicon, etc.
- `src/` – contains React source code, components, assets, and styles
- `node_modules/` – npm packages (ignored in Git)

## Installation

1. Install dependencies:

```bash
npm install
```

2. Run the project:

```bash
npm start
```

Open your browser at http://localhost:3000/ to see the project.
